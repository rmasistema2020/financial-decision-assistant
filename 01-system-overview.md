01 - System Overview
Project Name

Financial Decision Assistant

Purpose

The Financial Decision Assistant is a Python application that helps users evaluate their monthly financial situation by comparing income and expenses and generating a financial report.

Objectives
Practice Python programming.
Apply software architecture concepts.
Develop a modular application.
Practice clean code principles.
Build a professional portfolio project.
Functional Requirements

The system shall allow the user to:

Display the main menu.
Start a new financial analysis.
Enter monthly income.
Enter monthly expenses.
Calculate monthly savings.
Display a financial report.
Exit the application.
Non-Functional Requirements
The application shall be developed in Python.
The application shall use a command-line interface (CLI).
The project shall follow a modular architecture.
The source code shall be documented.
The project shall be version-controlled using Git.
Main Modules
Main Menu
Registration
Financial Calculations
Financial Report
Project Scope

The first version of the project will:

Perform one financial analysis.
Generate a financial report.
Return to the main menu.

Future versions may include:

Database integration.
User authentication.
Report history.
Charts and dashboards.
REST API.
Cloud deployment.
